# 📚 Topics Covered

## 1️⃣ Lambda Functions

### Definition

A lambda function is a small anonymous function that contains only one expression.

### Syntax

```python
lambda arguments: expression
```

### Examples

- Square of a Number
- Addition
- Even or Odd
- Maximum of Two Numbers
- Cube of a Number

---

## 2️⃣ map()

### Definition

The `map()` function applies a given function to every element of an iterable.

### Syntax

```python
map(function, iterable)
```

### Examples

- Square Numbers
- Capitalize Names
- Increase Student Marks

---

## 3️⃣ filter()

### Definition

The `filter()` function selects only those elements that satisfy a given condition.

### Syntax

```python
filter(function, iterable)
```

### Examples

- Even Numbers
- Passed Students
- Names Starting with A

# Key Takeaways

- Lambda creates anonymous functions.
- map() applies a function to every element.
- filter() selects elements based on a condition.
- Lambda works very well with map() and filter().
